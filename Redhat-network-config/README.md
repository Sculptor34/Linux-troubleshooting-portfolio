# Redhat network configuration

# Overview
- This project demonstrates how to solve network issue in Redhat 6.9.

# Problem
- The system displays network error during boot up

# Diagnosis
- The workstation unable to boot up normal startup
- The system cannot hard reboot
- Change the network card as suscpected hardware issue

# Solution
Steps:
1. Change the network card
2. Make some changes on network path
3. The network is restored
5. The hardisk able to boot up as linux environment

# Commands Used
- su
- mv
- reboot

# Result
- The workstation successfully to boot up as normal startup

# Notes
- This method should only be used for authorized system recovery.
