# Redhat configuration

# Overview
- This project demonstrates how to configure Redhat 6.9.

# Problem
- The system unable to boot up

# Diagnosis
- The workstation unable to boot up normal startup
- The system cannot hard reboot 
- Retry change slot SATA cable but it failed

# Solution
Steps:
1. Install ventoy into USB hardware
2. Take the Redhat ISO file paste in USB
3. Install Redhat ISO in new hardisk
4. Do the configuration step by step
5. The hardisk able to boot up as linux environment

# Commands Used
- su
- vi
- boot

# Result
- The workstation successfully to boot up as normal startup

# Notes
- This method should only be used for authorized system recovery.
