# path-to-inst is missing

# Overview
- This project demonstrates how to solve file missing in Unix.

# Problem
- The system displays path-to-inst file missing during boot up

# Diagnosis
- The workstation unable to boot up normal startup
- Retry hard reboot the system but it failed 

# Solution
Steps:
1. boot -a
2. Type Y
3. Press enter
5. The systems would create the path-to-inst automatically

# Commands Used
- boot -a
  
# Result
- The workstation successfully to boot up as normal startup

# Notes
- This method should only be used for authorized system recovery.
