# Unix resolution settings

# Overview
- This project demonstrates how to edit resolution settings in Unix.

# Problem
- The displays abnormal

# Diagnosis
- The monitor dispplays abnormal size
- Retry reboot and hard reboot but it still failed
- Swap the monitor and settings resolution monitor
- Found the root cause is systems in HDD displays abnormal

# Solution
Steps:
1. Reboot the workstation
2. Press stop + A and boot -s
3. Run these comand
4. Reboot one more time
5. The display is back to normal

# Commands Used
- boot -s
- reboot

# Result
- The workstation successfully to boot up as normal startup

# Notes
- This method should only be used for authorized system recovery.
